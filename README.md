# javascript-wdio-appium-nml-android-lambdatest

Boilerplate starter — no application is bundled. Point `APP_ID` (and, for Android, `APP_PACKAGE`/`APP_ACTIVITY`) at your own app, then adapt the test's `eyes.check()` calls to your app's own screens.

## Environment variables

- `APPLITOOLS_API_KEY`
- `LT_USERNAME`
- `LT_ACCESS_KEY`
- `APP_ID`
- `APP_PACKAGE`
- `APP_ACTIVITY`
- `DEVICE_NAME`
- `PLATFORM_VERSION`

## Run

```
npx wdio run ./wdio.android.lambdatest.conf.js
```
